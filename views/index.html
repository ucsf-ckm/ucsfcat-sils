Features and limitations

  • This takes permalinks from ucsfcat. For example: http://ucsfcat.library.ucsf.edu/record=b1305571~S0
  • It accepts http links or https links.
  • You can also just use the identifier (as it appears in the URL, so you might need to remove the final character if you got it from somewhere else). For example: b1305571
  • It will work for most/all of our print holdings and locally-licensed resources.
  • Unfortunately, it will not work for most/all shared holdings.
